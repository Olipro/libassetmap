Construct an instance of DirectoryMetadata.

The constructor will iterate through the directory passed and compute all necessary data required to archive it. Iteration is recursive and anything that is not a regular file will be ignored.

Parameters
hasherAny implementation satisfying IHasher.
compAny implementation satisfying ICompress.
entAn entry that references a valid, readable directory.

decltype(buckets) const& AssetMap::DirectoryMetadata::Buckets ( ) const
noexcept

Obtain a list of buckets and their entries.

This function provides a vector of vector of directory_entries. The indexable position of an outer vector element is its bucket ID. Every directory entry is guaranteed to be a regular file. Or at least, it was at the time of inspection - if you (or something else) change the on-filesystem data, things may break.

Returns
vector<vector<fs::directory_entry>>
