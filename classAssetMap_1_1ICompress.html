Compresses a range of data into an output buffer.

This function will compress a full sequence of data such that the output can be passed to an instance of IDecompress::Decompress and result in the same original input.

Precondition
It is implementation-defined whether src and dst may overlap. This is also true of passing any 0 lengths or a dstLen that is less than the result of CalcCompressSize. If dstLen is less than CalcCompressSize() there is no guarantee the compressed result contains the complete input data.
Parameters
srcA pointer to data at least srcLen in length.
srcLenThe length of data to compress.
dstA pointer to output
dstLen
Returns
The number of bytes written to the destination.

virtual bool AssetMap::ICompress::CreateDictionary ( std::filesystem::directory_entry  samplesDir)
pure virtual

Creates a dictionary from a valid, readable directory, if it can.

The directory is iterated recursively. How the dictionary is created is implementation defined. Typically a constructor will have been told a desired dictionary size. It is not an error for a dictionary to fail creation. This simply means the provided samples are not suitable for generating a dictionary.

Parameters
samplesDirA valid directory entry to use as samples.
Returns
a bool indicating whether dictionary creation succeeded.
