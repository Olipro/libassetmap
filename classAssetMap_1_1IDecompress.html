pure virtual

Decompress a previously compressed sequence of data.

Typically, when decompressing data, a call to CalcDecompressSize() will first be made to determine the destination buffer space needed.

Precondition
The data must have been compressed by a compatible ICompress implementation. It is implementation-defined as to whether src and dst may overlap and whether any size is permitted to be zero.
Parameters
srcThe buffer containing compressed data.
srcLenThe length of the source data.
dstDestination buffer for the output data.
dstLenMaximum number of bytes that can be written to dst
Returns
The number of bytes written into dst

Implemented in AssetMap::ZSTD.
