Constructs a MemMappedArchive for reading from an archive You should construct a separate instance of this class along with a separate IDecompress for each thread.

Postcondition
Anything that causes a change to the archive may result in undefined behaviour. This should usually fail at the point of trying to write to the IMemMapper as it should have been opened in read-only mode.
Parameters
filean instance of an existing, readable IMemMapper instance consisting of an entire complete archive.
compan instance of the compatible decompressor for the implementation used to compress the instance
hasheran instance of the hash implementation used to create the archive.

Constructs an instance for creating (optionally, reading) an archive.

Postcondition
If no decompressor or a decompressor incompatible with the compressor is provided, the behaviour of any archive read functions is undefined. All interface refs must remain valid for the duration of this instance's lifetime and any objects retrieved through this instance. file will be altered. Existing data will be destroyed.
Parameters
entA directory which will be recursed to create the archive.
hasherA hasher which will be used to determine bucketing.
fileAn IMemMapper which can (and will) be resized.
compAn implementation used to compress files.
decompAn optional decompressor should you wish to immediately read data back.
