AssetMap::MemMapper::MemMapper ( const std::filesystem::directory_entry &  file)
explicit

Constructs a MemMapper from a file path.

Precondition
If creating a file, the directory hierarchy must exist, it will not create it for you.
Postcondition
If the file exists and has a length > 0, it will be opened and mapped. Otherwise, it will be created/opened, but not mapped until a call to Resize() occurs. Even if mapped, the data is never writable after construction.
Parameters
fileA valid path to an existing file, or location to create one.
